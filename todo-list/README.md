## ğŸ“¦ Installation & Setup

Follow these steps to get the project running on your local machine:

1.  **Clone the repository:**

    ```bash

    cd todo-list
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

    ```bash
    npm start
    ```
